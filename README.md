# DUO PRIME CARE Website

A single-page website for DUO PRIME CARE pharmacy showcasing high-quality medicines, professional doctors, and premium healthcare services.

## Features

- Responsive single-page design that works on all devices
- Smooth scrolling navigation
- Elegant animations and transitions
- Contact form and newsletter subscription
- Modern, professional design with elegant typography

## Getting Started

To view the website locally:

1. Clone or download this repository
2. Open the `index.html` file in your web browser

## Customization

- Edit the text in `index.html` to update content
- Modify styles in `styles.css` to change the appearance
- Update functionality in `script.js` for custom behaviors

## Technologies Used

- HTML5
- CSS3 with responsive design
- JavaScript for interactive elements
- Font Awesome for icons
